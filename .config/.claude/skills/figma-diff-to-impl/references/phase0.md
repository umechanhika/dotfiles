# フェーズ0：技術スタック判定（サブエージェント用詳細指示）

サブエージェントはこのファイルの指示に厳密に従い、指定フォーマットの返り値だけを返す。

リポジトリの技術スタックを調査し、判定結果のみを返してください。
説明・感想・補足は一切不要です。

## 手順

1. リポジトリルートのファイル構成を確認する

2. 以下のファイルが存在すれば内容を確認する（存在するものだけでよい）
   - build.gradle / build.gradle.kts → Android / Kotlin
   - Package.swift / *.xcodeproj    → iOS / Swift
   - package.json                   → JS/TS系フレームワーク
   - pubspec.yaml                   → Flutter / Dart
   - *.csproj / *.sln               → .NET / C#
   - Cargo.toml                     → Rust
   - pom.xml                        → JVM系バックエンド

3. UIコンポーネント・画面ファイルのディレクトリ構成（src/, app/, lib/ 等）を確認し、
   使用フレームワーク・UIライブラリ・状態管理ライブラリを特定する

4. 文字列リソースの管理方法・カラー定義の場所を確認する
